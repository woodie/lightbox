# Photo Viewer

Display photos from flickr API.

## Test envisonment setup
```bash
npm install mocha
npm install chai
npm install mocha-jsdom
npm install jsdom
```

## Run tests
```bash
mocha
```

## Dev envisonment setup
Install Appengine (for Python)
https://cloud.google.com/appengine/docs/python/release-notes

## Run server
```bash
dev_appserver.py .
```
