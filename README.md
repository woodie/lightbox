# Photo Viewer

Display photos from flickr API.

### Test environment setup
* npm install mocha
* npm install chai
* npm install mocha-jsdom
* npm install jsdom

### Run tests
mocha

### Dev environment setup
* Install Appengine (for Python):
  https://cloud.google.com/appengine/docs/python/release-notes

### Run server
dev_appserver.py .
